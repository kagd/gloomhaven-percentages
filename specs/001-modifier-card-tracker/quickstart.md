# Quickstart: Modifier Card Tracker

## Prerequisites

- Node.js 22+
- npm

## Setup

1. Clone the repository
2. Navigate to project root
3. Install dependencies:
   ```bash
   npm install
   ```

## Development

Start the development server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

Build for production:

```bash
npm run build
```

## Usage

1. On first load, set up your player and monster decks by selecting card types and quantities.
2. Switch between player and monster views using the navigation.
3. Click on card stacks to draw cards.
4. Click the drawn pile to view and return cards.
5. Use the hamburger menu for reset options.

## Mobile Usage

The app is optimized for mobile devices. Use it on your phone during Gloomhaven games for real-time tracking.
